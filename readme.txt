== Description ==

This plugin allows you to subscribe an user to your Sendy list when he registers.

Sendy allows you to send emails with Amazon SES.
[You can download it here](http://sendy.co/?ref=1JYIN)

== Installation ==

1. Upload the synctosendy folder to your `/wp-coentent/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Sync to Sendy" tab and complete your information : sendy subscribe url (must end by /subscribe) and the List ID you want users to be subsribed to.
